# decode-labs-project3

Skill-Based Role Recommendation Engine
A career-path matching application that uses text vectorization through TF-IDF and Cosine Similarity to recommend optimal job roles based on a user's technical skill set. The project includes a terminal-based interface, an offline browser-based dashboard, and a production-ready FastAPI backend server.

Key Features
Advanced Matching Pipeline: Pre-processes text data and calculates similarity scores using computational vectors without requiring complex database architectures.

Dual Interface Design:

Terminal Mode: A lightweight Python script containing text-based analytics visualization.

Interactive Web Dashboard: A responsive HTML frontend interface featuring dynamic result-card rendering.

Hybrid Engine Toggle: The frontend features an environment switch allowing alternation between:

Local Mode: Utilizes a standalone client-side JavaScript mathematics engine.

API Mode: Routes live asynchronous API data payloads to the Python backend server.
